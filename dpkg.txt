Чтобы создать пакет .deb для Ubuntu с помощью dpkg, вам нужно выполнить несколько шагов. Вот краткое руководство:

Установите необходимые инструменты:
Убедитесь, что у вас установлены необходимые инструменты для создания пакетов. Вы можете установить их с помощью следующей команды:
bash


sudo apt-get install build-essential devscripts debhelper
Создайте структуру каталогов:
Создайте каталог для вашего пакета и перейдите в него:
bash


mkdir mypackage
   cd mypackage
Создайте подкаталоги:
Создайте необходимые подкаталоги:
bash


mkdir -p DEBIAN
   mkdir -p usr/local/bin
Добавьте файлы:
Поместите ваши исполняемые файлы или другие необходимые файлы в каталог usr/local/bin.


Создайте файл контроля:
В каталоге DEBIAN создайте файл control с содержимым, подобным следующему:




  
   Package: mypackage
   Version: 1.0
   Section: utils
   Priority: optional
   Architecture: amd64
   Depends: 
   Maintainer: Ваше Имя <ваш.email@example.com>
   Description: Краткое описание вашего пакета
    Длинное описание вашего пакета.
Создайте пакет:
После того как все файлы готовы, вернитесь в родительский каталог и выполните команду для создания пакета:
bash


dpkg-deb --build mypackage
Установите пакет:
После создания пакета вы можете установить его с помощью команды:
bash


sudo dpkg -i mypackage.deb
Теперь ваш пакет готов к использованию! Не забудьте проверить, что все файлы и зависимости указаны правильно.
